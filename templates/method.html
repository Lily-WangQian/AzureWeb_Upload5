Method

The Sustainability Report Alignment (SRA) uses Artificial Intelligence (AI) to compare documents based on keywords.

Database

The standard and regulatory documents have been researched by internet search and based on the current literature. The database contains 88 documents.

Data Processing

  • Data Cleaning: Stopwords are removed from the documents to reduce noise.
  • TF-IDF keywords extraction: TF-IDF is used to extract the top keywords from the text.
  • Contextual keywords extraction: A large embedding model is used to extract contextually relevant keywords.
  • Combination: TF-IDF and contextual keywords are combined for a richer representation.

Analysis

Similarity between documents is calculated using cosine similarity by representing documents as vectors in a high-dimensional space. A higher score indicates stronger alignment.
